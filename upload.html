How to "Upload" your Game

Since we are using GitHub Pages, you can upload games by simply adding them to your repository! Follow these steps:

Step 1: Create a Folder

Create a new folder in your project directory (e.g., games/your-game-name/).

Step 2: Add Game Files

Place your index.html, images, and scripts for the game inside that new folder.

Step 3: Update index.html

Add a new "Game Card" to the main index.html pointing to your new folder.

Step 4: Push to GitHub

Run these commands in your terminal:

git add .
git commit -m "Added a new game!"
git push origin main

Go to GitHub repository
